# gen_passwd.py

Tiny console password generator. Will generate passwords for you, currently suppoerted: [a-z,A-Z,0-9]

Requires python3.
No additional libs required.


Need 10 passwords, 16 chars each? Here you are:
gen_passwd.py -c 10 -l 16
or simply
gen_passwd.py 10 16
